Project Title: Website Content Management with Wordpress

For this project, I focused on creating a user-friendly website by following a structured UX process, including research, wireframing, prototyping, and user testing. The goal was to enhance usability and engagement through thoughtful interface design. I built the website using WordPress, selecting the Astra theme for its clean design and customization options. I developed layout ideas inspired by Tyler.com, ensuring a modern and intuitive user experience. To complete the branding, I designed a custom logo using LogoMakr.com, which I used as both the site icon and main logo.
